CareLink
Full-Stack NYC Resource & Community Hub
CareLink is a production-ready full-stack platform designed to bridge the gap between New Yorkers and essential social services. The application aggregates real-time NYC Open Data, curated news, and video guides into a centralized, high-performance dashboard with persistent user accounts and saved resources.
Built with a decoupled MERN architecture, CareLink emphasizes scalability, security, and real-world API integration while delivering a modern, accessible user experience.
